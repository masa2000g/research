# Excel自動生成プロジェクト

## プロジェクト概要
- 目的: 既存の「フォーマットが決まっている Excel ファイル」を、自動／半自動で生成できるようにする。
- 背景:
  - 会社やクライアントごとに Excel の仕様・フォーマットが微妙に違う。
  - AI に「それっぽく作って」と頼むだけだと、フォーマットが崩れたり、仕様変更への追随が難しい。
  - 手作業で毎回作るのは面倒なので、テンプレート＋自動生成の仕組みを探りたい。
- 技術候補:
  - Python（`openpyxl`, `pandas`, `xlsxwriter` など）
  - あるいは他の言語・ツールでもOK（要検討）。

## 当面のタスク（叩き台）
1. 現状の Excel フォーマット収集
   - 実際に使っている Excel サンプルを集める（会社別・用途別）。
   - 「絶対守りたいルール」と「柔軟に変えてよい部分」を言語化する。

2. 自動生成の手段リサーチ
   - Pythonライブラリや既存ツール（テンプレートエンジン）を調査。
   - 「テンプレートファイル＋入力データ」から安全に出力できる方法を探す。

3. 小さな PoC（検証）
   - まずは１つのフォーマットを Python 等で自動生成するミニスクリプトを作る。
   - OpenClaw 経由でスクリプトを回したり、梓からパラメータ投入できる形を検討。
