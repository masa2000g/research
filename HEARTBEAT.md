# HEARTBEAT.md

## やること（優先順）

1. **タスク管理**: `tasks/YYYY-MM-DD.md` が今日分なければ作成。前日の未完了を引き継ぎ、issues/ から今日やれることを拾う。
2. **inbox巡回**: `inbox/daily/` の直近3日分を確認。新しいキャプチャがあれば `inbox/staged.md` を更新する。
3. **イシューチェック**: `issues/` の open なイシューを確認。進められるものがあれば1つ進める。
4. **プロジェクト進捗**: 活動中のプロジェクトがあれば、READMEの進捗を更新する。

5. **便利機能リサーチ（随時）**: 会話の中でマイクの課題やニーズを感じたら、Web検索で関連ツール・API・手法を調べて、具体的な提案として1つだけ伝える。cronの定期提案(水・土14:00)とは別に、タイミングが良ければ随時やってよい。

## 判断

- 上記すべてチェックして何もなければ `HEARTBEAT_OK`
- 何かあればマイクに軽く報告（1-2行で）
- 深夜帯 (23:00-07:00) は HEARTBEAT_OK で静かにしておく（タスク作成・inbox巡回は裏でやる）
