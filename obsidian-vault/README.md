# Masa Obsidian Vault

This folder is the shared "vault" for:

- Obsidian (Masaのノートアプリ)
- OpenClaw / 梓（ここからノートを読んだり書いたりする）

## 基本ルール

- このフォルダ配下は **Markdown（.md）ファイル** を基本にする
- テーマごとにサブフォルダを作ってOK（projects/, ideas/, logs/ など）
- 大事なメモ・知識・議事録は、できるだけここに集約する

## 想定フロー

1. Obsidian でこのフォルダを Vault として開く
2. 日々のメモ・アイデア・ナレッジをここに書く
3. OpenClaw（梓）に「obsidian-vault の中から○○を探して」「この内容でノートを作って」と頼む
