# 🧩 因数分解バトル！

中学生が因数分解を楽しく学べるブラウザゲームです。

## 起動方法

`index.html` をブラウザで開くだけで遊べます。

```bash
open index.html
```

サーバー不要。HTML/CSS/JS のみで動作します。

## ゲーム概要

式を因数分解して、全5ステージをクリアしよう！

### ステージ構成

| ステージ | 内容 | 例 |
|---------|------|-----|
| 1 | 共通因数くくり出し | `6x + 12 = 6(x + 2)` |
| 2 | 完全平方式 | `x² + 6x + 9 = (x + 3)²` |
| 3 | 平方の差 | `x² − 25 = (x + 5)(x − 5)` |
| 4 | x² + bx + c 型 | `x² + 5x + 6 = (x + 2)(x + 3)` |
| 5 | ボス戦（ミックス・高難度） | 上記すべて + 係数付き |

### ゲームシステム

- **HP**: 3つ。間違えるか時間切れで1つ減る。0でゲームオーバー
- **コンボ**: 連続正解でボーナス得点
- **制限時間**: 1問25秒
- **ヒント**: 正解・不正解どちらでも解き方のヒントが表示される
- **復習**: 結果画面から間違えた問題を復習できる
- **ランク**: 正答率に応じて S〜D のランク評価

### 対応環境

- PC ブラウザ（Chrome, Firefox, Safari, Edge）
- スマートフォン（レスポンシブ対応）
