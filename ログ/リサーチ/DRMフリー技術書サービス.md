# リサーチログ: DRMフリー技術書サービス

## 目的
- 「最初からDRMフリーPDFとして買える技術書／プログラミング本」の入手先を把握する。
- 将来的に、NotebookLMなどの学習用ノートブックに読み込ませる教材ソース候補とする。

## 現時点で有力な候補（一般論）

※2026-04 時点の一般的な情報に基づく。

1. 技術同人・個人出版系
   - 技術書典、BOOTH など
   - 多くが PDF 版を直接販売（DRMフリー）

2. Leanpub
   - 著者主導の技術書プラットフォーム。
   - 通常、購入すると PDF / EPUB / MOBI などの DRMフリー版をダウンロード可能。

3. 一部の技術出版社・直販
   - 例: Pragmatic Programmers（PragProg）、Manning など
   - かつては PDF ダウンロード＋DRMフリーが一般的だった。
   - 現状の詳細は各社サイトの購入ページで要確認（DRM 方針が変わることがある）。

4. 大学・研究機関のオープン教材
   - OER（Open Educational Resources）として公開されている PDF 教材。
   - 大学の講義ノートや教科書が DRMフリーで配布されているケースもある。

5. DRMフリー書店のリスト
   - Libreture の "DRM-free bookshops" リスト
     - 様々なジャンルの DRMフリー書店のカタログとして使える。

## まだ要確認のポイント
- 各出版社（O'Reilly, No Starch, Manning など）が 2026年現在も DRMフリーPDFを直販しているかどうかは、個別確認が必要。
- 日本語の技術書で DRMフリーPDFを公式に提供している出版社・ストアがどこまであるかは、別途深掘りが必要。

## 今後のアクション候補
- 具体的に「この本を買いたい」というタイトルが決まったら、その本について個別に「DRMフリーの公式版があるか」を調べる。
- NotebookLM連携を前提に、「PDFを安全にアップロードして自学習に使えるソース」候補をリストアップしていく。
